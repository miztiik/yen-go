## Decision Record

| Decision           | Value                                                     | Rationale                                   |
| ------------------ | --------------------------------------------------------- | ------------------------------------------- |
| Backward compat    | Thin façade re-exports `enrich_single_puzzle`             | Zero caller changes                         |
| Dead code          | Remove after extraction                                   | Dead code policy                            |
| Test imports       | Update to target new modules                              | Tests should exercise the module they cover |
| AI-Solve paths     | Extract as private async functions + `EnrichmentRunState` | SRP: each function = one code path          |
| `_uncrop_response` | Move to `query_builder.py`                                | Cropping module = forward + reverse         |
| Config lookups     | Full consolidation into `config_lookup.py` across 4 files | DRY + SRP                                   |
| State carrier      | `EnrichmentRunState` dataclass before extraction          | Minimum data contract for clean extraction  |
| Notify pattern     | Attach `notify_fn` to `EnrichmentRunState`                | Avoids parameter noise                      |
