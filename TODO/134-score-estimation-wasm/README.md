# Yen-Go

A fully offline Go (Baduk/Weiqi) tsumego puzzle platform. Practice life-and-death problems anytime, anywhere — no internet required.

## Play Now

**🎮 [Play online at yen-go.github.io](https://yen-go.github.io)** — no installation needed!

### Features

- 🎯 **Daily Puzzles** — New puzzles every day across 9 difficulty levels (Novice → Expert)
- 🎮 **Puzzle Rush** — Timed challenges to test your speed
- 🏆 **Achievements** — 30+ badges to unlock
- 💡 **Progressive Hints** — Three levels of help when you're stuck
- 📱 **Mobile-Friendly** — Touch support with responsive design
- 🔌 **Offline-First** — Works without internet as a PWA
- ♿ **Accessible** — Full keyboard navigation support

### Controls

| Input     | Action           |
| --------- | ---------------- |
| Click/Tap | Place a stone    |
| `H`       | Show hint        |
| `S`       | Show solution    |
| `R`       | Reset puzzle     |
| `←` `→`   | Navigate puzzles |

Your progress is saved locally in your browser — no account needed.
